# Snake Game 项目规划

## 1. 项目概述

开发一个画面精美、操作流畅的贪吃蛇游戏，支持简体中文界面。

## 2. 技术选型

- **开发语言**: Python 3.8+
- **GUI框架**: Pygame (跨平台、成熟稳定)
- **打包工具**: PyInstaller (支持多平台构建)

## 3. 功能规划

### 3.1 核心功能
- [x] 蛇的移动控制（上下左右）
- [x] 食物生成与吃到食物增长蛇身
- [x] 碰撞检测（墙壁、自身）
- [x] 分数系统
- [x] 游戏开始/暂停/结束界面

### 3.2 游戏特性
- [x] 经典贪吃蛇操作方式（方向键控制）
- [x] 速度随分数增加而加快
- [x] 最高分记录
- [x] 美观的图形界面

### 3.3 系统功能
- [x] 日志系统（保存到游戏目录的 log 文件）
- [x] 简体中文界面

## 4. 界面设计

### 4.1 游戏区域
- 网格化游戏区域 (40x30 格子)
- 每个格子 20x20 像素
- 游戏区域带有边框装饰

### 4.2 颜色方案
- 背景色: 深色 (#1a1a2e)
- 蛇身: 渐变色 (绿色系)
- 食物: 红色带发光效果
- 文字: 白色/浅色

### 4.3 界面元素
- 标题: "贪吃蛇"
- 分数显示: 当前分数 / 最高分
- 操作提示
- 游戏状态提示

## 6. 构建与发布

### 6.1 GitHub Actions 配置
- 触发条件: 创建版本标签 (v*)
- 构建目标: Windows, Linux, macOS
- 输出格式:
  - Windows: .exe
  - Linux: .tar.gz
  - macOS: .tar.gz
- 校验文件: checksums.txt

### 6.2 版本命名
- 使用语义化版本: v0.1.0, v0.2.0, ...

## 7. 开发计划

1. **Phase 1**: 基础功能实现
   - Pygame 环境搭建
   - 蛇的基本移动
   - 食物生成与碰撞检测

2. **Phase 2**: 游戏完善
   - 分数系统
   - 速度递增
   - 游戏状态管理

3. **Phase 3**: UI 美化
   - 界面设计
   - 动画效果
   - 简体中文支持

4. **Phase 4**: 系统功能
   - 日志系统
   - 最高分记录

5. **Phase 5**: 打包发布
   - PyInstaller 配置
   - GitHub Actions 配置
   - 发布测试
